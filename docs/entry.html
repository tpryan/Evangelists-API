count
remote numeric count ( string source="", string author="", any startDate="", any endDate="", any q="" )

Returns a count of records matching input criteria. Useful for paging interfaces.

Output:
Parameters:
   source: string, optional, source - The external source of the content. A list of posible sources is available via listSources()
   author: string, optional, author - The username of the creator of the content. A list of posible sources is available via listAuthors()
   startDate: any, optional, startDate - The begining of a date range to filter content. If ommitted, count will count records back to the earliest record
   endDate: any, optional, endDate - The end of a date range to filter content. If ommitted, count wil count records forward to the latest record
   q: any, optional, q - A search term which will be matched against the TITLE and DESCRIPTION of the content.

get
remote Evangelists.cfc.content[] get ( string source="", string author="", any startDate="", any endDate="", any q="", numeric offset="0", numeric maxresults="20", string orderby="id asc" )

Returns a list of records matching input criteria.

Output:
Parameters:
   source: string, optional, source - The external source of the content. A list of posible sources is available via listSources()
   author: string, optional, author - The username of the creator of the content. A list of posible sources is available via listAuthors()
   startDate: any, optional, startDate - The begining of a date range to filter content. If ommitted, get wil retrieve records back to the earliest record
   endDate: any, optional, endDate - The end of a date range to filter content. If ommitted, get wil retrieve records forward to the latest record
   q: any, optional, q - A search term which will be matched against the TITLE and DESCRIPTION of the content.
   offset: numeric, optional, offset - A record number to start the search from. Useful for paging.
   maxresults: numeric, optional, maxresults - The count of records to retrieve at one point. Useful for paging.
   orderby: string, optional, orderby - The order in which to retrieve the records.
