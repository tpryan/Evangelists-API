Evangelist API Docs

Getting Started

To use the API start with the base url:

http://yourdomain.com/evangelists/api/index.cfm/

Then append modifiers to it

First comes what you are looking for, or the method:

get|count|listAuthors|listSources

From there you add modifiers to filter your search:

ModifierformatDefault
authora comma delimited list of authorsall
sourcea comma delimited list of sourcesall
daterangea range of dates in the format yyyymmdd-yyyymmddall
formatxml or jsonxml
offset and maxresultsNumbers in the format offset|maxresults0|20
qa search term in QuotesNone

Examples

Get all records for terryr in json format

http://yourdomain.com/evangelists/api/index.cfm/get/terryr/json

Get a count of all records from vimeo for March 2011in xml

http://yourdomain.com/evangelists/api/index.cfm/get/vimeo/20110301-20110401

Get a list of all records since the end of March

http://yourdomain.com/evangelists/api/index.cfm/get/vimeo/20110331-

Get a list of 5 records for ptrani and rstewart from vimeo and youtube before march 2011 in json

http://yourdomain.com/evangelists/api/index.cfm/get/rstewart,ptrani/vimeo,youtube/-20110301/0|5/json

Get a list of 5 records for ptrani and rstewart from vimeo and youtube before march 2011 in json that are about "blackberry"

http://yourdomain.com/evangelists/api/index.cfm/get/rstewart,ptrani/vimeo,youtube/-20110301/0|5/"blackberry"/json
